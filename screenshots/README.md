## Como tirar screenshots no Android Studio:

1. Execute o aplicativo no emulador ou dispositivo
2. Navegue até a tela desejada
3. Use o botão de screenshot na barra lateral do emulador
4. Ou use Ctrl+S (Windows/Linux) ou Cmd+S (Mac)
5. Salve as imagens nesta pasta

## Tamanho recomendado:
- 1080x1920 pixels (portrait)
- Formato PNG ou JPG
